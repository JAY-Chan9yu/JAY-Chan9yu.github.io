Frequently Asked Questions (FAQ)

How many events can I register?
You can register unlimited events. Add as many anniversaries, birthdays, exams, and more as you want.
My widget is not updating.
Try removing the widget by long-pressing it, then add it again. If the problem persists, please restart the app.
I want to change the character.
Tap the character icon in the event edit screen to choose from various characters.
I'm not receiving notifications.
Please check if notifications are enabled in Settings > WigglyDay > Notifications.
Can I backup my data?
We currently support iCloud backup. Please enable iCloud backup in your device settings.
